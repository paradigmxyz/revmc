Struct EvmWord

Source 
#[repr(C, align(8))]
pub struct EvmWord(pub(crate) [u8; 32]);
Expand description

A native-endian 256-bit unsigned integer, aligned to 8 bytes.

This is a transparent wrapper around [U256] on little-endian targets.
