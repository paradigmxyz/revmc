Enum Value

pub enum Value {
    Generic(u64),
    I8(i8),
    U8(u8),
    I16(i16),
    U16(u16),
    I32(i32),
    U32(u32),
    I64(i64),
    U64(u64),
    F32(f32),
    F64(f64),
}
Available on crate feature cranelift only.
Expand description

The value of an entry on the DWARF stack.

This corresponds to the DWARF DW_OP_convert operation.
