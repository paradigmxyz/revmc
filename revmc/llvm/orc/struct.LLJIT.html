Struct LLJIT

pub struct LLJIT {
    jit: *mut LLVMOrcOpaqueLLJIT,
}
Available on crate feature llvm only.
Expand description

An ORC JIT.

Manages the memory of all JIT’d code and all modules that are transferred to it.

See the ORCv2 docs.
